- At least one **image** in `images/`
- Embedded **YouTube** video on Home
- Home has a working **mailto** link

## Make It Live (GitHub Pages)
1) New GitHub repo → upload all files.  
2) Settings → Pages → Source: Deploy from a branch; Branch: `main`, Folder: `/ (root)` → Save.  
3) Wait for build; your Pages URL will appear in Settings.

A `.zip` is provided for easy upload.
